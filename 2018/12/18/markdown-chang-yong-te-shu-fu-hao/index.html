Markdown 常用特殊符号

发表于 2018-12-18 | 分类于 Markdown
字数统计: 494 | 阅读时长 ≈ 2

Markdown由于其简洁的排版风格受到很多码字员的喜欢，由于其本身支持HTML语法，所以在markdown中加入HTML支持的语法会使文章更具表现力，在这里介绍一些常用得HTML特殊符号。

符号说明对应编码英文怎么说
