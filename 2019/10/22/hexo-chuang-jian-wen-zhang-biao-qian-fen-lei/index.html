hexo 创建文章、标签、分类

发表于 2019-10-22 | 分类于 hexo

1、创建文章

1.1在hexo下创建一个新的文章
 hexo new "文章名称"

生成后会提示你文件路径，一般在hexo/source/_posts下

1.2文章基本设置
---
title: yum安装docker
date: 2019-10-22 19:01:24
comments: true #是否可评论
toc: true #是否显示文章目录
categories: "云服务器" #分类
tags:   #标签
    - docker
    - yum
---

2、创建标签

2.1创建标签页面
hexo new page tags
2.2基本设置

source/tags目录下index.md文件

title: tags
date: 2019-10-22 19:03:11
type: "tags"
layout: "tags"

3、创建分类

3.1创建分类页面
hexo new page categories
3.2基本设置

source/categories目录下index.md文件

title: categories
date: 2019-10-22 19:07:45
type: "categories"
layout: "categories"
